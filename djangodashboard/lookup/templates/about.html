About Me...

Djashboard allows you to define api calls and select some key values in a json config file. Once you do that, and after you map a category value to a color and after you map the category to a description, you will have added a related widget -- a Bootstrap jumbotron -- to the dashboard reflecting that API call. You can configure as many widgets as you'd like for any given dashboard. {% endblock %}
